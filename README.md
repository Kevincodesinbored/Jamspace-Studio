# 🎶 JamSpace - Studio Booking System

JamSpace is a comprehensive music studio rental platform designed to help musicians find and book rehearsal or recording studios quickly and efficiently.

## 🚀 Key Features
- **User Authentication**: Secure Login and Registration system implementing industry-standard password hashing (bcrypt).
- **Studio Exploration**: Detailed studio browsing, including hourly pricing and community-driven ratings.
- **Booking System**: Integrated reservation engine with real-time availability validation to prevent double bookings.
- **Shopping Cart**: Flexible cart system allowing users to manage multiple reservations before finalizing the checkout.
- **Notification System**: Automated notification engine for booking confirmations, schedule reminders, and session completion alerts.
- **Review & Rating**: Post-session feedback system allowing users to provide star ratings and detailed reviews.
- **Automated Cron Job**: Simulated backend process to send automatic reminders for bookings scheduled for the next day.

## 🛠️ Tech Stack
- **Language**: PHP 8.x
- **Database**: MySQL / MariaDB
- **Frontend**: HTML5, CSS3 (Modern UI), JavaScript (ES6)
- **Icons**: FontAwesome 6.5.0
- **Server**: XAMPP / Apache
